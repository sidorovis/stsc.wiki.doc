##### stsc.algorithms(common)
 * Input
```
{ {no_input},
e -> [String: open by default, high, low, close, value]}
-> DoubleSignal
```
 * ListOfDoubleAdapter
```
{ {1: ListOfDoubleSignal},
e -> [Integer: I - index of double into list 0 by default]}
-> DoubleSignal
```
 * Output
```
{ {1: DoubleSignal or SideSignal or IntegerSignal},
e -> [-SubAlgoName-]}
-> DoubleSignal.
```
